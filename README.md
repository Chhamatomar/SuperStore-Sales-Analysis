# 📊 Superstore Sales Analysis & Interactive Dashboard

## 🚀 Project Overview

This project presents a comprehensive analysis of a retail Superstore dataset to uncover key business insights and trends. The analysis is performed using **Exploratory Data Analysis (EDA)** techniques, followed by the creation of a fully interactive **Excel Dashboard**.

The objective is to transform raw data into actionable insights that can help improve business performance and decision-making.

---

## 🎯 Business Objectives

* Evaluate overall **sales and profit performance**
* Identify **top-performing categories and sub-categories**
* Analyze **regional sales distribution**
* Detect **loss-making areas**
* Understand **customer purchasing behavior**

---

## 🛠️ Tools & Technologies

* **Microsoft Excel**

  * Pivot Tables
  * Pivot Charts
  * Slicers
  * KPI Cards
* **Data Cleaning**
* **Exploratory Data Analysis (EDA)**

## 📂 Dataset Information

* Dataset: *Superstore Sales Dataset (USA)*

  * Order Date

  * Profit
  * Category & Sub-Category
  * Region
  * Customer Details

## 🔍 Data Analysis Workflow

1. Data Cleaning & Preprocessing
2. Handling Missing Values & Duplicates

6. Dashboard Development

## 📈 Dashboard Highlights

* 🔹 Interactive Filters (Slicers)
* 🔹 Key Performance Indicators:

  * Total Sales
  * Total Profit
  * Total Orders
* 🔹 Visual Insights:

  * Sales by Category
  * Profit by Region
  * Monthly Sales Trend
  * Top Performing Products

---

## 💡 Key Insights

* High sales do not always lead to high profit
* Certain sub-categories consistently generate losses
* The West region contributes significantly to revenue
* Seasonal trends impact sales performance

This project demonstrates the power of Excel in transforming raw data into meaningful business insights. It highlights key performance metrics and provides an interactive way to explore sales trends.
